Minia explore les dynamiques internes d'un MLP entraîné sur MNIST.
Actuellement, le projet se focalise sur l'interprétabilité et le monitoring. L'enjeu est de définir des métriques pertinentes pour sonder les deux matrices de poids du réseau, afin de quantifier l'importance relative des neurones et d'évaluer la qualité de l'apprentissage en temps réel.
